# Verkefni 5: Gæðaeiginleikar fyrir umönnunarkerfið Stoð

## Höfundar og áfangi

* **Áfangi:** HBV301G Verkfræði kröfugreiningar
* **Höfundar:** Brynja Kristín Bertelsdóttir og Óli Þorbjörn Guðbjartsson

## Um verkefnið

Þetta verkefni snýst um að skilgreina og rökstyðja mikilvægustu gæðaeiginleika fyrir umönnunarkerfið **Stoð**. Verkefnið krefst þess að við veljum **5 ytri** (sem snúa að notanda/rekstri) og **3 innri** (sem snúa að þróun/viðhaldi) gæðaeiginleika.

Fyrir hvern eiginleika er skrifuð ein gæðakrafa með rökstuðningi. Að lokum eru **3 pör** hugsanlegra árekstra greind og forgangsröðuð.

## Innihald geymslu (Repository)

* **`answers/quality-attributes.md`:** Tafla með öllum 8 gæðakröfunum og rökstuðningi.
* **`answers/planguage-reliability.md`:** Formleg útfærsla á gæðakröfunni fyrir "Áreiðanleika" skv. PLanguage sniðmáti.
* **`answers/conflicts.md`:** Greining á 3 árekstrapörum og rökstudd forgangsröðun.
